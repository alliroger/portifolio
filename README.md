Este é o meu portfólio pessoal, desenvolvido para centralizar meus projetos, habilidades e experiências no ecossistema de desenvolvimento de software.
Ele foi construído com foco em performance, design responsivo e Clean Code.

Sobre o Projeto
O objetivo deste site é servir como um cartão de visitas digital. Ele apresenta uma interface moderna com animações em gradiente e um sistema dinâmico de exibição de projetos usando JavaScript puro.

Principais funcionalidades:
Design Responsivo: Adaptado para dispositivos móveis, tablets e desktops.

Filtro de Projetos: Sistema de "Mostrar Mais" que carrega novos projetos dinamicamente.

Interface Interativa: Efeitos de hover e animações @keyframes para uma melhor experiência do usuário.

Clean Code: Estrutura de arquivos separada (CSS, JS, Assets) seguindo boas práticas de organização.

Tecnologias Utilizadas
Para este projeto, utilizei o trio fundamental do Front-end:

HTML5: Estruturação semântica.

CSS3: Estilização avançada, Flexbox e animações personalizadas.

JavaScript (Vanilla): Manipulação do DOM para interatividade.

Font Awesome & DevIcons: Ícones para redes sociais e tecnologia.

Google Fonts: Tipografia personalizada (Poppins e Raleway).
